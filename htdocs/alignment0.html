Locate the crystal

Move mph to 10 mm above the liquid sample cell by moving the manipulator plate-to-plate distance 108.5mm. Set the manipulator height zero and limits.
set mph 0
set_lm mph -9 5
Move chamber height to make the beam pass through the chamber at 103mm below the top 13 1/4" flange. Move motor sx to find the crystal center. Move motor sx to place the sample crystal center at the beam center.

Motor mpr alignment

Move motor sh down to cut the beam intensity by 90%.
dscan sh

Scan motor mpr to maximize detector counts. set mpr 0
dscan mpr
set mpr 0
set sh 0
